Misha Theme

Version 1.0

Installation

  1. Copy the theme folder to your BlogEngine files in the Custom > Themes folder.
    Warning: Don't rename the folder and don't change "misha" to "Misha". it's case sensitive.
  2. Then in the admin panel, navigate to your Themes and click on the active button.

Customization

  1. In the admin panel, Navigate to Custom > Themes.
  2. On the activated theme, click on the customize icon.

You can set your logo file address and social media accounts and other things.

Take a look at this example:

Slideshow

  1. In the admin panel, Navigate to Custom > Widgets.
  2. Drag the "PostList" Widget into the "Widget Zone (Slider)" and then save it.
  3. Click on the Edit icon and then enable the "display thumbnail" and "display date" and then save it.
